# Neural networks from scratch

Neural networks implemented using Numpy.


## Features

- Automatic differentiation
- **Layers**
    - Linear
    - 2D convolutions
    - 2D max-pooling
- **Activations**
    - Sigmoid
    - Softmax
    - Tanh
    - ReLU
- **Losses**
    - Binary cross-entropy
    - Categorical cross-entropy
- **Optimizers**
    - SGD
    - Adam
- Extremely slow
